Inline post editor inspired by the simplicity of Medium.com editor

== Description ==

Medium.com WYSIWYG editor clone. Uses contenteditable API to implement a rich text solution
This is a WordPress version of the Medium Editor: https://github.com/yabwe/medium-editor

= How to use the editor =
   * Click on the "Edit" button thats added bellow each post/page
   * Select any line to toggle the options toolbar
   * Click "Save" to update and "Cancel" to discard the changes

== Installation ==

FROM WP ADMIN: Install from the WordPress.org respository by going to Plugins > Add New and search for Medium Editor. Install & Activate.
BY UPLOADING: Upload and extract the zip file and just drop the contents in the wp-content/plugins/ directory of your WordPress installation and then activate the Plugin from Plugins page.

== Frequently Asked Questions ==

= How to edit posts? =

Below each post/page there is a "Edit" button added. When you click on this button you will be able to edit the text area for the selected post.

= Any Settings? =

None, Medium Editor works out of the box. Just install & activate

= Can I change the toolbar design? =

Only if you are familliar with css. :)

= I want more! =

We're considering adding Image Upload, Buttons, Lists, Embed, Tables and Custom HTML to the toolbar. Let us know in the comments which one would you like us add first.

= Which browsers are supported? =

Medium Editor requires modern browsers to work properly.
Currently supports Chrome, Firefox, Safari, IE and Edge
